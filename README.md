## Technical Exercise for Anthropic

### Versions used
Java 8

Junit 5

### How to Run
Create a new java project in eclipse with java8 run env

copy src/ and its files there

resolve junit5 by adding it to build path

We can do it by clicking on red line errors in `FlightServiceTest.java`


Now run it as java application on right clicking `AirlineApp.java`
